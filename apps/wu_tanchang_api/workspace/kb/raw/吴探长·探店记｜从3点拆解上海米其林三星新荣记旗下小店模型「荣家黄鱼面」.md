# 吴探长·探店记｜从3点拆解上海米其林三星新荣记旗下小店模型「荣家黄鱼面」



#### 探店动机



如今的大环境，什么样的人会去吃一碗358元/份的东海小黄鱼面呢？竟然还排队！反正我是舍不得，关键还不是在宴请的包房里，而是在一家100来平的面馆里。吴探长挑了周中，和美食大咖一起来到上海浦东前滩太古里这家“荣家黄鱼面馆”——米其林三星新荣记旗下子品牌，我们一起来看看餐饮老手们是怎么做的？



#### 核心概述



本次探店将围绕“消费场景定位”、“菜单结构设计”、“产品体验细节”三个维度，拆解这家开在顶级商圈、背靠米其林三星品牌的小面馆，如何用“大餐饮轻餐化”的创新模式，在百平米空间里实现高客单、高坪效与高品牌势能的平衡。



#### 正文拆解



**1、消费场景定位：大餐饮轻餐化，快餐社交化**



主打一碗高价值感的黄鱼面，定位“改善型快餐”，满足一人食与少人化轻社交消费场景。在饭点时段，适合三三两两的同事小聚，也适合来太古里逛街的游客。不同人群的需求场景，就意味着要在就餐环境、价格和菜品结构上做好对应匹配——“七分选对位，三分做差异”。



能看出这家黄鱼面馆和普通开放式的走量快餐店不同，就餐环境注重一定的私密度，装修风格偏日式。要知道如今这样的中小店型，最值得借鉴的体验设计就是日式的omakase或居酒屋的交付互动形式：把重油热炒和粗加工、洗碗间都放在暗厨，把干净清爽的煎煮厨房和切配冷菜间开放为明档，增强体验品质感。



100来平方，50来个客位：7张4人台、4张2人台、2条大板1人食，围绕具有烟火气的中央区域开放厨房，整个小店的气就聚起来了，人的体验感自然会加分，满意度就上去了。



主品系列定价从78元、138元到368元三个阶梯。要是你请客或者犒劳自己一下，有高价招牌的138元和368元黄鱼面来支持；也有常规特色走量款主打性价比的78元海鲜面。价格层次带出了客群宽度，营收就上去了。



要是光卖一碗面，一来卖不上价格，在这样的高租金场子开，模型就不成立；二来和纯快餐没区别，客单价很难突围。所以能看出这家店有重餐饮背景和思维，打了个错位竞争：主要照顾到高客单的少人化聚餐需要，以招牌黄鱼面做主品驱动，强关联品牌定位记忆点；关键要发展周边副品，搭配卤味、点心、小菜，偏向打造mini版轻社交请客消费场景。



再加开在A类市级商圈太古里商场，米其林三星新荣记出品背书——请你吃一碗面也不会坍台吧？里子面子都有了。这就是“大餐饮轻餐化，快餐社交化”的创新模式。



**2、菜单结构设计：主品驱动，副品延展，工艺复用**



产品结构背后就是生产模式，生产模式决定出品效率和成本结构。



先看主品黄鱼面分2款：小黄鱼和大黄鱼的食材区别，黄鱼面的汤底也不同，那就需要熬制2锅汤。



1. 限量款：选用新鲜东海小黄鱼，368元/份

2. 特色招牌：选用新鲜大陈岛大黄鱼，138元/份

    

海鲜面分2款：特色原味海鲜面和姜汤米面，口味有区别，但可以复用一款汤底，辅材和6个调味料也一样复用。



面分2种：手工粗鲜面和台州地方细米面可选，备货成本会高，点单效率会低。



菜单中主食除了主品面系列，还有牛肉面、鸡丝面、卤鹅饭等扩品；周边副品除了饭前小菜和饭后点心，主推卤味以鹅为主材展开（鹅头、鹅肝、鹅心等部位）现切现售，最近还推出和米其林三星新荣记店里一样的一道家烧大黄鱼。



这样的小店产品结构，在工艺上却横跨了热炒、煎、蒸、煮、熬、烤、凉拌——其实就是一个重餐饮的生产配置了，只是单个工艺设备和面积减少而已，综合成本不低，还需要进一步优化调整。



**3、产品体验细节：汤底见真章，体验定口碑**



一叶一世界，更何况这一碗面呢？这里一碗面出品时间要10分钟。清水煮面到热汤，放料环节——吃海鲜面讲究个汤底和料。料分主材和辅材：贝壳类、虾干、蛋块、香肠片、干肉丝、菜叶、肉制品等，确实很丰富。这意味着口感层次丰富，边吃边捞，满足感影响满意度。其实食材成本不高，但体验很好。点黄鱼面就是有黄鱼肉，汤底也不同，这一点值得点赞。



汤底方面，海鲜面的汤底我以为是纯黄鱼熬制的，想多了。据服务员告知，海鲜面的汤和2款黄鱼面汤底不一样，在黄鱼底汤的基础上加了稍低成本的猪大骨熬制。一上来闻有腥味，吃起来不觉得腥。黄鱼面由于腥，调味上必须放上黄酒和姜粉，强化鲜度上再放些少许的盐、味精和鸡粉。



这次体验下来，口味有点寡淡，鲜度没出来，而且浓度感再上去一些就更好了。总体两个熟人朋友边吃边聊，环境和服务还是不错的，价格还行，不算便宜也不算贵。



#### 商业洞察总结



荣家黄鱼面的成功，在于将米其林三星的“品牌势能”成功下沉到百平米小店模型中。它用“改善型快餐”重新定义了面馆的消费场景，用“主品阶梯定价\+副品延展”构建盈利结构，用“明档体验\+日式布局”提升空间价值。其核心壁垒不是单一产品，而是新荣记多年积累的供应链、品牌信任与出品标准在小店模型中的精准复用。
