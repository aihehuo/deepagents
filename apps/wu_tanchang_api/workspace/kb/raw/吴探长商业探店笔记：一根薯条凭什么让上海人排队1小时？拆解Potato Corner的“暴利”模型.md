# 吴探长商业探店笔记：一根薯条凭什么让上海人排队1小时？拆解Potato Corner的“暴利”模型



#### 探店动机



在上海吴江路，一家名为Potato Corner的薯条专门店最近火得一塌糊涂。不就是一个卖土豆的小档口吗？竟然有人愿意排队1个小时，只为吃上一口刚出锅的“卷卷薯”。这背后究竟是网红的短暂狂欢，还是藏着什么高人一等的商业逻辑？吴探长带着满腹疑问，在一个周日下午亲临现场，试图从选址、产品、营收、品牌四个维度，拆解这个源自菲律宾、全球拥有1700家门店的“薯条帝国”是如何炼成的。



#### 核心概述



本次探店，我们将透过热闹的排队表象，深入剖析Potato Corner的“小店大连锁”模式。重点解读其如何利用“黄金地段\+极致坪效”的选址策略截流，通过“形状\+味型”的排列组合拉高客单价，以及依靠“去厨师化”的标准化SOP实现快速复制。在小吃赛道极度内卷的当下，Potato Corner用一根薯条证明了：只要模型跑得通， humble的食材也能卖出黄金价。



#### 正文拆解



**1\. 选址策略：流量为王，截杀核心商圈**

开小吃档口，位置定生死。Potato Corner深谙此道，首店便落在了上海国际商圈南京西路的吴江路。



这里是绝对的流量高地：



- **交通枢纽**：紧邻地铁2号、12号、13号线南京西路站，汇聚了全市乃至全国的游客。

- **客群精准**：周边高密度写字楼林立，白领和年轻人是薯条类零食的天然受众。

- **氛围加持**：整条街餐饮氛围浓厚，饭点人气爆棚。

    

选址在这里，意味着自带巨大的自然流量，无需过多的额外引流成本，无论是周中还是周末，都能保证源源不断的客流。



**2\. 产品策略：一颗土豆的“排列组合”艺术**

选品即经营。Potato Corner的店面仅约25平米，却通过极致的SKU组合，把简单的生意做厚了。



- **形态创新**：除了普通薯条，它推出了卷卷薯（Loopy Fries）、表情包薯、薯格和薯粒（Tater Tots）。这种形态上的微创新，瞬间拉开了与普通快餐店薯条的差距，增加了产品的趣味性和价值感。

- **味型丰富**：芝士、洋葱、BBQ、热辣烧烤等4种核心味型，覆盖了咸、甜、辣、香多种口味偏好。

- **高效出品**：店内仅需4人（收银、炸制、撒粉、打包），配合全自动炸炉，几分钟即可出餐。这种“去厨师化”的操作，不仅降低了人力成本，更保证了口味的绝对标准化。

    

**3\. 营收模型：高客单\+高翻台，坪效惊人**

在现场观察发现，Potato Corner的营收能力远超预期。



- **客单价设计**：虽然起步价仅18元，但大多数消费者会选择38元的双拼套餐。通过“大小份\+多口味”的引导，轻松拉高了客单价。

- **营收测算**：周日下午2点，叫号已到300多号。若按平均客单38元计算，仅这一时段的营收就已破万。若营业至晚9点半，日营收预估可达2万左右。

- **租销比健康**：月营收预估60万，即便租金高达5万，租销比也仅为10%左右（通常餐饮业警戒线为15%\-20%）。这种极佳的盈利模型，为其后续的快速扩张提供了充足的弹药。

    

**4\. 品牌力：神形兼备的“洋品牌”降维打击**

品牌力的打造，Potato Corner做到了“神形兼备”。



- **视觉识别**：品牌名直白（Potato），IP形象可爱，黄绿配色极具食欲和辨识度。虽然吴探长认为其符号感（如那个大大的“\&amp;”）还有提升空间，但整体形象已足够鲜明。

- **价值锚定**：土豆本是廉价食材，但Potato Corner通过“菲律宾网红品牌”、“全球1400\+门店”、“东南亚风味”等标签，成功实现了品牌溢价。

- **排队效应**：始终保持在50米以上的排队长度，本身就是最好的广告牌。这种“羊群效应”不断吸引着路人加入排队大军，形成良性循环。

    

#### 商业洞察总结



Potato Corner的成功，不是靠运气，而是靠对“小吃模型”的极致打磨。



它没有试图教育市场，而是顺应了年轻人对“重口味、高热量、即食”零食的喜爱；它没有陷入价格战，而是通过“卷卷”形态和“摇摇”吃法创造了差异化价值；它没有依赖大厨，而是用标准化的设备和流程锁定了利润。在消费降级的当下，它告诉我们：越是看似不起眼的品类，越藏着巨大的机会，关键在于你能不能把“简单”的事情做到“极致”。



#### 思考题



Potato Corner目前依靠“排队效应”和“网红属性”在上海站稳了脚跟，但随着门店的加密和竞品的模仿（如SH4KE、Putoto等），这种依赖“新鲜感”的模式该如何维持？当排队消失后，它又该如何通过产品迭代或会员运营，留住那些挑剔的“薯条脑袋”？
