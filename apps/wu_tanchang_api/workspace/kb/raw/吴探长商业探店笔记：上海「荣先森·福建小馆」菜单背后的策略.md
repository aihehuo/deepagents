# 吴探长商业探店笔记：上海「荣先森·福建小馆」菜单背后的策略



#### 探店动机



快过春节了，一句“年年有余”扎心了，今年没有“余”，那就去吃一顿大黄鱼吧，补偿一下年度心情。话说福建人会吃鱼，更会卖鱼，吴探长在上海大宁久光发现这家米其林一星遇外滩旗下的福建小馆「荣先森」，上海有8家店，主打地道闽南菜。特别吸引我的地方是他家菜单，小小菜单里确有大文章。「餐饮有道，老吴有解」看完我以下3点的拆解，你就清楚了如何正确的对菜品做排兵布阵，让进店客人听你指挥，帮客人学会点、能快点、点得下去，满足了他人的需求，顺带出自己的利润。



#### 核心概述



本次探店将从定主品推爆品、选大将地方军上、定价格定客单这3点展开品牌咨询项目的实施细节分享，视频比较长啊，建议你先收藏起来。



#### 正文拆解



**1、定主品，推爆品**

因为一道菜记住了一家店，这道菜必须够硬，无论从食材和工艺都代表了自家的最高水平。主打一道古法鲜黄鱼汤，用的是大黄鱼之乡的福建宁德黄鱼（出水冷链，锁鲜直达），5年卖了60万\+份，还入选《中餐特色美食名录》等等支撑点，真是铆足了劲大力推招牌菜，实现每座必点，从菜单的占有率和排列顺序看得出是“菜老大”的地位。定价98一锅，是汤也是菜，眼见为实，整条大黄鱼现烫现吃，仪式感拉满，要知道这些动作和信息传达直指目标客群，无论你是家庭客、同事小聚还是情侣、朋友，都适合来一份，此菜通杀！

这还没完，光主品还不行，容易被孤立，可能因为口味、价格被弃选。照顾价格敏感和口味偏好的客群，来了个一拖三的金字塔式主品方阵，就像织成一张网。一材多菜，同样大黄鱼在口味上延伸酱油水黄鱼烧年糕和陈年老萝卜蒸大黄鱼，一烧一蒸分开烹饪工艺，就像皇亲国戚一样，2款定价128元，高出主品98元30%，后厨直接出品，要知道这全是利润啊！再来，你要是嫌大黄鱼价格高，还有32小份62大份低配版小黄鱼兜底。要知道就这一页的主品系列，通常你就会点一道，畅销的一定是中配版98这一招牌菜，因为性价比排在你面前，没有理由点不下去，高明吧！



**2、选大将，地方军上**

主品招牌菜点好了，靠的推荐引导能力，接下来就交给客户自由搭配咯，no！no！继续指导，全程要陪跑，不是活人而是菜单，接下来就是人气热销菜。这什么概念，是众多食客选出来的帮你避雷了，可不像是主品那样是我推荐给你的哦。我们细看菜单这第二页，首先要从食材结构来看，要互补主品系列的鱼类，有猪腊肉、猪脚、牛排、鸭、海鲜和传统小吃，烹饪工艺煎炒、炖煮、炸，和主品菜做互补，就是君和臣之间的搭配关系。

从点单逻辑到了这三页，基本主帅和大将基本确定了，接下来就兵团上阵了，以地方传统特色菜作为大使，福建家常的经典菜就是福建菜符号，定位了也定格了。要知道传统菜很难做出差异，也很难突破高度，对连锁发展来说标准也是道坎，这就是镣铐，怎么跳也会有一定的束缚，只好出来再进去，承续传统底色，聚焦价值创新。比如当下主推的时令菜，头水紫菜，一个主材扩展了5道菜，不同做法不同口味，吃得好可以现场卖走。

地方兵团，不仅仅是地方符号，还要担当填满一桌菜的功能，从食材结构上也不能和人气热销菜用的主材冲突，点了牛排肉就不会再点牛排了，但可以点炒黄牛肉，荣先森这点又把握住了。福建特色菜、经典菜、时令头水紫菜、砂锅焗小海鲜这几类里选材很好避开了这一点。这样的关系就如中医理论里讲“君臣佐使”，平衡之道，每个分类结构都有自己角色，各司其职，排列有序，构成整个产品方阵。



**3、定价格，定客单**

横轴是产品定价，纵轴是产品数量，约55个菜品（饮品、酒水除外），中位数50元以下低价区菜品数量多，给人感知便宜不贵，多点几样。50\-100之间，重点突出60\-69元菜价的数量，100以上菜品很少，只有4道大菜，锚定品质效应。这样的菜品组合，排兵布阵得出人均不超过140元，2\-3人为主，桌单在300元左右，适合轻社交、家庭小聚，以上推算，个人观点仅供参考，主要想给大家一点启发。



#### 商业洞察总结



一家店是个大模型，菜单就是一家店的小模型，紧密的连接着前台消费需求场景和后台生产交付。荣先森通过“金字塔式”的爆品策略锁定核心利润，利用“君臣佐使”的食材互补逻辑提升连带率，最后通过精妙的价格锚定控制客单价。菜单更关系着如何服务他人，顺带自己，是打开用户满意度和店铺利润之门的一把关键钥匙。



#### 思考题



荣先森通过“菜单工程学”极大地提升了点单效率和客单价，但这种高度结构化的菜单设计，在面对追求“随机性”和“探索感”的Z世代消费者时，是否会显得过于刻意？在餐饮连锁化率越来越高的今天，你认为“标准化的菜单逻辑”和“个性化的厨师创意”哪个才是品牌长青的关键？欢迎在评论区留下你的看法。
