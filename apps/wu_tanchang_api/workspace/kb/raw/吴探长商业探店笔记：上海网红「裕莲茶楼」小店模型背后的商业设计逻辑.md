# 吴探长商业探店笔记：上海网红「裕莲茶楼」小店模型背后的商业设计逻辑



#### 探店动机



“茶楼”一词，让我想到老舍笔下的《茶馆》，影片中“裕泰茶馆”的场景，有歇脚儿的、喝茶的、商议事情的、相面的、做买卖的人都混迹于此，茶馆成了反映民国时期的一种社交文化的载体。如今，茶楼不见楼，茶楼不仅仅卖茶，因为一款焦糖可颂蛋挞而走红，还需要每日线上预定，疯了！疯了！他到底是怎么定义卖茶这门生意的？这背后具体有哪些地方值得我们思考和学习的？跟着吴探长一起来到上海大宁久光这家不到50平方的中岛店“裕莲茶楼”一探究竟。



#### 核心概述



本次探店将分享以下3个方面的洞察，看完你就明白了一个不到50平方的中岛店，是如何通过产品策略、爆款制造和组合销售，实现高效营收和品牌传播的。



#### 正文拆解



**1、产品策略**

这么一个小小的店，从制作到售卖，可以看出来他实际上定位的不是体验空间，而是在茶楼里所提供的食物。我的理解他只是借用了一个民国时期的茶楼概念和场景营造，名正言顺的整合吃的、喝的，这一点也算是自洽。



要是你要是正儿八经的约人去茶馆，是不是得每人泡上一壶茶，来一盘瓜子坚果、一碟果肉蜜饯、几小块精致的中式糕点，还得有个“雅间”？这里只有食物，没有雅座，价值定位上放弃空间体验价值，放大食物价值提供，给他个赞。



跟着我的镜头看现场陈列：茶罐、蜜饯，桌面排着圆嘟嘟的煮茶电陶炉、透明柜里放着袋装曲奇、泡芙等，灯箱都是一些茶饮。做法就是将茶叶预制冲泡，再根据订单现场冲调，基茶的种类基本是通识的乌龙、绿茶、红茶，选用价值感高且香气很突出的凤凰单枞、茉莉、滇红、大红袍等。传统价值感的茶和现代价值感的食材一融合，就是一款茶饮，如加入椰奶、柠檬、枣泥、牛乳、桃胶等调和。看了现场设备和操作工序，在做法上并没什么特别的技术和新花头，区别于传统的功夫泡茶或者现场茶萃取工艺，这样能实现出品效率高，成本低，卖你20来块钱，消费者感知其实够了。



**2、制造爆款**

前店后场，现做现售，特别是热食的蛋挞这个品类特性，刚出烤箱的最好吃，暖呼呼香喷喷，大小适中，方便走食，也可以多买几个分享装打包带走。可以看出“裕莲茶楼”并没有把钩子产品押注在茶上，而是一款创新蛋挞“焦糖可颂蛋挞”，又是一次跨界组合，真服了！



用可颂面包表胚替代传统蛋挞底胚，可颂面包中要加入了大量的黄油，口感酥脆，层次分明，外表金黄，饼皮酥脆伴有黄油香，用的还是ECHIRE艾许品牌黄油；蛋挞里的蛋液加入焦糖、牛奶一起烤制，里面的奶香浓郁、嫩旺旺的，外皮焦甜，这样的口感组合，有几个抵御得住的？一个12元，价格其实也还好，被食客们定义“殿堂级”，为产品研发点个赞。



因为一款招牌产品，记住了一个品牌；因为一款引流产品，搭配了其他饮品的销售。现场可以买现制品，陈列的成品蜜饯、曲奇、茶叶等都可以顺带买走，有了零售的思维，打开了消费场景，更打开了喝茶的体验场景。并非一定要提供堂食，一人走食，休闲零食、商品零售化，这样轻量店型，启动和复制起来也容易。



**3、组合销售**

实话讲，我到现场一看，第一认知是个现代茶饮店，是红茶公司等茶饮专门店没啥区别。再看看有果肉蜜饯、蛋挞、曲奇等，感觉很杂，但好像这样的组合也存在关联。从用户需求角度看，你喝一杯茶饮多数是下午茶，搭配甜点的泡芙、曲奇、蛋挞都合情合理。看来这样的搭载销售是有效的，看到现场有人组合的买，自然单均就拉上去了。



要是你再买点零售商品茶叶和蜜饯的话，那营收结构就随着这样的“大融合”就搞大了。如今在这样的经济大环境下，多品类组合，一口价，打出性价比，用套餐化思路是拉动销售营收的好方法。我想“裕莲茶楼”在推出套餐上可以多做文章，将这些现有的品进行不同组合搭配，引导前端消费，其实这样做并不改变后端生产能力的变化。



#### 商业洞察总结



裕莲茶楼的成功在于“茶饮\+烘焙\+零售”的复合模型创新。它避开了传统茶馆的重资产投入，用一款“殿堂级”焦糖可颂蛋挞作为超级流量入口，带动高毛利的茶饮和零售商品销售。其核心竞争力在于将传统茶文化进行轻量化、零食化改造，用爆款思维打破品类边界，实现了小店型下的高坪效和高复购。
