# 吴探长商业探店笔记：九十葉凭什么让上海人排队1小时？拆解抹茶赛道的“浓度”生意经



#### 探店动机



最近，上海的抹茶圈被一家名为“九十葉”的店彻底点燃了。打着“甄选世界抹茶”的旗号，这家源自静冈的抹茶专门店在新天地首店开业后，迅速引发了“抹茶脑袋”们的狂热追捧，排队时长一度飙升至1小时。在冰淇淋赛道早已拥挤不堪的上海，九十葉究竟有何魔力，能让年轻人心甘情愿地付出时间成本？吴探长特意选在一个周末前往现场，试图从品牌、产品、运营、竞争四个维度，拆解其背后的经营之道。



#### 核心概述



本次探店，我们将透过排队的热闹表象，深入剖析九十葉的商业逻辑。重点解读其如何通过“浓度分级”的差异化策略精准切分客群，利用“一帅九将”的菜单模型最大化坪效，以及通过高度标准化的SOP实现快速复制。在Venchi、Luneurs等强劲对手的包围下，九十葉的突围不仅是一次品类的胜利，更是一场精细化运营的教科书式演示。



#### 正文拆解



**1\. 品牌与招牌：用“浓度分级”重新定义抹茶冰淇淋**

餐饮经营讲究“三牌合一”，即品牌、招牌、餐牌的统一。九十葉的招牌产品无疑是“抹茶冰淇淋”，但它最聪明的一点在于，将抹茶这一单一品类做出了“颗粒度”。



不同于市面上千篇一律的抹茶口味，九十葉将抹茶浓度细分为5个层次（如2号、3号、5号等）。这一策略极其高明：



- **拓宽客群**：入门级消费者可以选择清淡的浓度，而资深“抹茶控”则能追求极致的苦与醇。

- **增加营收**：多一个浓度选择，就多一个成交机会，且不会增加后端供应链的复杂度。

- **建立专业度**：像手冲咖啡分浅烘、中烘、深烘一样，浓度分级瞬间拉高了品牌的专业壁垒，让消费者觉得“这钱花得值”。

    

**2\. 菜单结构：“一帅九将”的排兵布阵**

如果说一家店是一个大模型，那么菜单就是这家店的小模型。九十葉的菜单设计深谙“一帅九将”之道。



- **主帅**：抹茶冰淇淋是绝对的核心，占据展示柜最显眼的位置（通常是上排绿色区域），负责输出品牌心智和主要流量。

- **大将**：下排陈列的清酒（獭祭）、开心果（西西里）、薄荷巧克力、咸焦糖、馥芮白、生巧等6种口味，虽然色彩诱人，但更多是起到辅助和衬托作用。它们的存在是为了满足同行伙伴中“非抹茶爱好者”的需求，防止因一人不爱吃抹茶而流失整单生意。

    

此外，盛器分为蛋筒和杯装，进一步拉宽了价格带和消费场景。这种精细化的产品组合，极大地提升了点单转化率和客单价。



**3\. 运营交付：去“人化”的标准化SOP**

在实地观察中，九十葉的运营效率令人印象深刻。这家约20平米的中岛店，动线设计极为流畅。



- **S型排队**：利用围栏引导顾客排成S型，既制造了人气旺盛的视觉效果，又预留了足够的点单缓冲期。

- **极简岗位**：点单收银、挖球交付、打包出品，每个环节仅需一人。

- **低门槛依赖**：核心竞争力被押注在“食材”和“配方”上，而非员工的手艺。员工只需经过简单培训即可完成标准化的挖球动作，无需像法式甜品师那样依赖长期经验。

    

这种“重食材、轻人工”的模式，不仅保证了出品的一致性，更为后续的连锁化复制扫清了最大障碍——人才瓶颈。



**4\. 竞争格局：在巨头缝隙中寻找生态位**

上海的冰淇淋市场强敌环伺。



- **Venchi**：主打巧克力\+冰淇淋，走的是“零售\+堂食\+礼品”的高端路线。

- **Luneurs**：以法式面包\+冰淇淋切入，强调烘焙与冰淇淋的结合。

    

九十葉没有选择与它们正面硬刚“复合业态”，而是选择了“专门店”的打法，将抹茶这一细分品类做到极致。它不卖面包，也不卖昂贵的巧克力礼盒，只专注做好这一杯抹茶冰淇淋。这种“单点突破”的策略，使其在激烈的市场竞争中成功撕开了一道口子，占据了“正宗抹茶”的心智高地。



#### 商业洞察总结



九十葉的走红，并非偶然，而是对“细分品类”进行“深度开发”的必然结果。



它没有试图讨好所有人，而是通过浓度分级精准服务了不同层级的抹茶爱好者；它没有盲目扩充品类，而是通过“一帅九将”的菜单逻辑稳住了基本盘；它没有依赖大厨，而是通过标准化流程实现了高效交付。在消费回归理性的当下，九十葉证明了：只要在一个针尖大的领域里，把产品做深、把体验做透，依然能爆发出惊人的商业能量。



#### 思考题



九十葉目前依靠“抹茶专门店”的势能迅速扩张，但随着门店数量的增加，如何避免品牌陷入“只有网红属性，缺乏复购粘性”的陷阱？在新鲜感过后，它又该如何挖掘抹茶品类的更多可能性，以维持长期的生命力？
