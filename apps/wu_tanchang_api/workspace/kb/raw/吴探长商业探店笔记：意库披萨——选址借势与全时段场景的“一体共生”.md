# 吴探长商业探店笔记：意库披萨——选址借势与全时段场景的“一体共生”



#### 探店动机



要是你在上海市中心只开一家小店，你会选什么品，选在哪里开，怎么开？



有人说在披萨界，只有意大利的那不勒斯披萨和其他比萨2种，竟然有如此大的自信啊！上海这家刚开不久的网红那不勒斯比萨店，据说他是亚太区最会得奖的披萨品牌，刚荣获亚洲第二佳、连续三年荣获比萨大红虾三瓣奖（相当于米其林三星的荣誉）。



在华丽的光环下，我更关注的是他在开店方面的经营策略，特别是选址定位、产品策略和场景设计这3方面，如何自成一体的。吴探长选了周末饭点后时间来到落位在上海市中心A类商圈的Bottega意库的上海首店一探究竟，是他继北京2家店之后的第三家店，从底层逻辑上展开对他的关键成功元素以及产品核心要素的洞察。



#### 核心概述



意库披萨上海首店的成功，并非单纯依靠“世界排名”的光环，而是一场关于 **“高维定位与场景渗透”** 的精准战役。它通过 **“借势选址”** 锁定高净值人群，利用 **“全时段菜单”** 覆盖多元社交需求，并以 **“剧场式场景”** 营造沉浸式意式氛围。这三者环环相扣，构建了一个既具有国际高度又具备本土烟火气的商业闭环。



#### 正文拆解



**1\. 选址定位：借势A类商圈，锁定“高净值”流量池**

店址落在市级商圈环贸商场和襄阳公园对面的甲级写字楼嘉华中心配套的底商1楼，写字楼出来对面就是，大大的通透店铺展面很显眼。点位并不是像边上沿街商铺位西式轻食moka摩卡，沿路的能见度其实并不高，但这丝毫不影响他的火。



这就很考验品牌的营销获客能力了，依据这样的点位分析，建圈谁会是他的第一批目标顾客？锁定的固定客群来自这栋楼上的高级打工人群，对西式餐饮接受度高，社交频次高，购买力强。周中的午餐、晚餐都适合少人化社交，时段从brunch早午饭到凌晨12点，人均近200元。目测这里面积有近300平方，80来个客位数，2人位占一半，2\-4人就餐为主，也有1人食和4\-6人客位区的桌型布局，有独立的分男女卫生间，能满足多元化的社交场景；周末生意更好，客源结构来自附近网红街襄阳路、陕西路citywalk以及毗邻的环贸商场和襄阳公园逛的流动人群。



看来选址一定要学会“借”：借目标客群扩展潜在客群，借流动人群聚客热点来补充自己客群，以及借商圈定位支撑品牌定位和定价，借场地面积来支持多元化消费场景等。



**2\. 产品策略：全时段经营的“菜单工程学”**

当店址店型确定了，接下来就是产品策略的设计了，从菜单品类结构上要满足全时段经营和多场景产品搭配。



主品是那不勒斯披萨，系列分为经典款像水牛奶酪披萨、玛格丽特等，以及多款特色风味披萨，一共有近20个sku；披萨通配12寸，定价区间89\-159元，制作工艺延续那不勒斯的披萨特点，24小时面团发酵、手工拍底、450度高温大窑炉60秒快速烘烤，0号面粉，饼皮轻薄酥脆、饼边有标志性的黑色豹纹斑点等。



从功能区布局上看，一进门口对面就是开放式披萨窑炉、边上拍底和配料台；右边区域给到了酒品区，红白葡萄酒、鸡尾酒等；最里面的后厨主要是制作副品，有前菜的意大利火腿拼盘、色拉、例汤、炸物小食等，主食类有意大利面和特色饭类，餐后甜品等配置。



产品结构和操作功能分区很清晰，一目了然，把制作披萨的主品和全时段饮品前置，很多外籍服务员，再结合软装和音乐来烘托，晚上8点后灯光会调暗些，酒吧氛围就出来了，通过这些元素组合搭配呈现出一个具有时尚个性、轻松有活力的就餐体验。



**3\. 场景设计：定位一体化的“氛围剧场”**

说到上面选品逻辑，有一个原则，要紧跟着选址定位和店型的消费场景来设计，不要想当然，这3个元素即是独立，又是相互咬合关系，相辅相成，相互衬托和表达，是定位一体化呈现的设计概念。



意库的空间设计不仅仅是装修，更是一个“氛围剧场”。从进门看到的定制3吨重火山石烤炉，到80年代复古迪斯科工业风格的软装，再到夜晚动感的音乐和特调鸡尾酒，每一个环节都在强化“那不勒斯的热情”这一主题。这种设计让顾客在享受美食的同时，也完成了一次短暂的心理度假。



我们再看一个同样街铺的快餐披萨小店模型，就在边上南昌路上joe‘s pizza，也是个网红店，约80个平方的沿街铺位，他的主品是车轮大披萨，按片售卖，选用大的烤箱而不是窑炉，轻堂食。从产品结构上看，披萨sku不到15款，炸物5款，只有1款红酒，10来款饮品，人均约50元，同样也能满足全时段经营的搭配。这背后是什么在指导这些产品组合数量和产品制作、以及消费体验场景上和价格期待上差异设计？

#### 商业洞察总结



意库披萨的案例告诉我们，**“高客单价”并非高不可攀，关键在于是否提供了匹配的“价值感”与“场景感”。**



它没有选择流量最大的沿街铺位，而是选择了 **“目的性消费”** 更强的写字楼底商，通过 **“借势”** 周边的高端流量和商圈调性，精准筛选出了高净值客群。在产品上，它没有局限于单一的披萨，而是通过 **“全时段\+全品类”** 的菜单结构，将生意从午餐延展到了深夜酒吧，极大地提升了坪效。



**选址是骨架，产品是血肉，场景是灵魂。** 三者缺一不可，只有当它们高度统一时，品牌才能形成独特的磁场，吸引并留住那些对品质有要求的消费者。
