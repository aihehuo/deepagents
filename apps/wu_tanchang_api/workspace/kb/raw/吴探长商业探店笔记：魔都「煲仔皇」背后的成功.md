# 吴探长商业探店笔记：魔都「煲仔皇」背后的成功



#### 探店动机



我想大家都感到了当前经济在下行吧，但对基本品的要求却在往上走，不将就，要吃好的牛奶、好的咖啡、新鲜好吃的饭菜等；来年虽有众多不确定性，但今天这一顿必须确定，眼见为实，米来自哪里？腊肠是哪个品牌？是不是现做的？接下来，快餐店怎么做才能赢得消费者的青睐？带着这些问题，吴探长抽个周中饭点时间，来到上海市级商圈环球港B2楼的快餐层。据说能在这个场子活下来都是有真功夫的，看到一排的快餐店有小满手工粉、马记永兰州拉面还有小杨生煎，只有这家煲仔皇在排队，上前一探究竟。



#### 核心概述



本次探店将分享以下3点洞察，看完你就明白了一个100平方的快餐店型该如何合理地设计产品结构和生产模式，包括如何有效地与门前路过的潜客沟通，提升进店转化率。



#### 正文拆解



**1、产品定位**

菜单越薄，利润越厚，这背后的设计要讲究双赢，从工艺和主材上下功夫。从煲仔皇看到主打煲类，生产工艺一致，煲分2大系列：广式的煲仔饭和啫啫煲。为了满足生产管理和用户体验，在煲的盛器上做了区分。从菜单上看到主材集中在牛肉、鸡肉、生肠、排骨等常规通识食材，一材多菜，基本可以复用，批量采购。为了丰富SKU选择，通过不同酱汁做出口味区别，搭配些周边的汤品、卤味小吃、爽口小菜和饮品等。再从陈列上就能直观看到有皇上皇的品牌腊肠和泰国的茉莉香米，要知道用品牌原材料不仅仅是支撑品质溢价，更是保障品控稳定。



**2、需求场景**

我们再从产品满足人群的需求场景上看，要知道主食驱动的快餐多数是集中在午市，晚市很难做，比如面馆。煲仔饭可以满足一人食，也可以几个人点上几份啫啫煲，再搭配些副品，不仅仅是适合办公一族的工作餐，也适合晚市小聚；目测现场2人位居多，定位少人化的轻餐饮\+轻社交。这样的空间和品类组合即抓住了上班人群的工作餐，又适合周末来商场逛的游客人群，显然是以菜品驱动型的快餐模型。有肉有菜有碳水，人均40来块钱，能满足三三两两的同事、同学、朋友和小家庭等多元化客群需要。



**3、用户沟通**

快餐本身讲究一个快，快决策、快出餐、翻台快，引导潜客快进店是第一步，要用货架思维。一家店就是一条街铺上的产品，明档、展陈、海报、门头、口号等等都是产品包装元素，重点是利用视觉符号和口号词语来传达差异卖点和信任点，帮助新客做决策，要让客户像坐滑梯一样滑进店。跟着我的镜头看：煲仔皇作为品牌名植入品类名，我家是做“煲仔饭和啫啫煲”，我家卖点“煲煲现做有锅巴”，我家招牌“腊味四宝煲仔饭”年销200万份，我家特色是“茉莉香米”一年只产一季，传达优质，腊肠腊肉由“老字号皇上皇定制”，鲜嫩滑鸡是“谷饲喂养新鲜宰杀”。眼见为实，明厨亮灶，一个煲一个煲都是现做让你看到，陈列的大米你也能摸摸，菜单价格直接亮相，“全面武装”地打消你的踩坑顾虑，勾起你想尝试欲望，你还有什么理由不进店呢？



#### 商业洞察总结



煲仔皇的成功在于将“品质快餐”做到了极致。通过“现做现煲”的明档模式，解决了快餐“预制菜”的信任危机；通过“煲仔饭\+啫啫煲”的双品类驱动，打破了传统快餐的时段限制；通过品牌食材的背书，构建了高性价比的认知。其核心壁垒在于将复杂的广式烹饪标准化，同时保留了“锅气”这一灵魂体验。
