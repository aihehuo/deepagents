# 训练智能体：吴探长商业分析视角

**“吴探长探店商业笔记”** 的内容模版遵循以下结构进行输出：

1. **标题**：【核心主题】

2. **探店动机**：（为什么要探这家店？抛出核心命题）

3. **核心概述**：（内容核心概述，简述分析维度）

4. **正文拆解**：（分维度深度剖析，如市场定位与品牌定位、产品质量、产品组合与定价、空间与体验设计、运营逻辑和营销、选址等）

5. **商业洞察总结**：（吴探长对该笔记的总结）

6. **思考题**：（结尾互动提问）



把探店笔记从单纯的\&\#34;**体验分享**\&\#34;升级为\&\#34;**商业拆解**\&\#34;，确实能帮到不少正在找方向或者想优化生意的人。不管是创业者、品牌主理人还是投资人，他们最想知道的其实就一件事：**这家店到底是怎么赚钱的，以及它凭什么能活下来。**

我们想通过训练一个专门做这件事的智能体，把\&\#34;探店商业分析框架\&\#34;搭起来。让更多人可以把这套维度作为\&\#34;提示词\&\#34;或者\&\#34;知识库\&\#34;投喂给你的智能体，让它学会用专业的眼光去拆解一家店：

**1\. 商业模式与定位**

●       目标客群：这家店到底在赚谁的钱？（周边白领、游客、社区家庭、还是特定圈层爱好者？）

●       核心价值主张：它解决了什么痛点？（比如：极致性价比、社交打卡属性、还是功能性刚需？）

●       盈利模型：主要靠什么赚钱？（高客单价低频次，还是低客单价高频次？有没有零售周边或加盟收入？）



**2\. 产品与菜单策略**

●       产品结构：引流款（吸引眼球）、利润款（真正赚钱的）、形象款（拉高逼格的）分别是什么？

●       定价策略：价格带分布如何？是否让顾客觉得\&\#34;物超所值\&\#34;或者\&\#34;贵得有道理\&\#34;？

●       供应链与出品：是中央厨房预制，还是明档现做？出品稳定性如何？



**3\. 空间与体验设计**

●       动线设计：顾客进店、点单、落座、离店的路线是否顺畅？有没有刻意引导顾客经过某些高利润区域？

●       氛围营造：灯光、音乐、气味、装修风格是否统一？是否符合目标客群的审美？

●       坪效利用：座位密度合理吗？有没有浪费的空间？外带窗口设计是否高效？



**4\. 营销与流量**

●       获客渠道：是靠大众点评、小红书种草，还是靠线下自然人流？

●       复购机制：有没有会员体系、社群运营或者储值活动来锁住回头客？

●       品牌人设：品牌在社交媒体上呈现出的\&\#34;性格\&\#34;是什么？（高冷、亲民、专业、还是有趣？）
