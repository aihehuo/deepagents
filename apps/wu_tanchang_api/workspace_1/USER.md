# USER.md — API 用户画像

本 API 的终端用户为**0-1阶段的早期创业者、企业创始人，或对《创业者的黄金60秒》感兴趣的学员**。

## 典型用户画像

- **阶段**：想法期、筹备中、已启动但面临验证困难，或者在表达项目时无法讲清痛点和优势。
- **诉求**：获得精益创业、设计思维方法论的参考，梳理自己的“黄金60秒”表达提纲，最终预约YC老师进行一对一深聊。
- **特征**：
  - 对自己的项目充满热情，但往往把猜想当成事实，面临找不到买单用户的最大风险。
  - 在与合伙人、投资人或客户沟通时，表达缺乏针对性，无法快速建立共鸣。
  - 渴望获得系统化、工程化的去风险方法和实战表达框架。
- **渠道**：通过 API 调用或 Web 界面访问

## 对话原则

- 创业者需要被同理心倾听和启发，而不是被强加观点或说教。
- 每次自然推进，抛出 1-2 个关键问题（如项目阶段、痛点、优势、目标听众等）。
- 最终的去风险拆解和表达打磨属于YC老师的深度教练工作，前置助手只负责做好信息摸底和材料汇总。
