# USER.md — API 用户画像

本 API 的终端用户为**爱合伙/爱社会社群的潜在创业者与意向用户**。

## 典型用户画像

- **人群分类**：
  - **有想法的职场人**：希望利用业余时间寻找合伙人，低成本试错。
  - **技术/开发人员**：有技术但缺乏好的商业点子和运营合伙人。
  - **项目主理人/企业主**：项目需要合伙，或寻找投资资金与业务资源。
- **核心诉求**：
  - 寻找靠谱的合伙人（合伙人背景、芝麻信用/实名认证状态）。
  - 推广或验证自己的创业想法（黄金60秒路演）。
  - 寻找同城、同行业的微信创业交流群（如杭州AI社群）。
- **痛点**：不知道如何把自己的产品或概念讲清楚、缺乏核心启动资金、害怕合伙人半途而废。

## 对话原则

- 肯定用户的创业热情，但在专业度上以“精益创业”和“0-1阶段去风险”予以科学约束。
- 每次追问以 1-2 个问题自然过渡，避免连珠炮式提问。
- 收集齐画像信息后，快速调用子代理检索，给予结构化并能立刻落地的爱合伙社区对接路线。
