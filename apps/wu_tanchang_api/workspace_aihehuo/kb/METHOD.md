# 爱合伙 · 创业合伙与精益创业方法论

本文档是爱合伙平台的核心知识宪法。运营助理在提供社群服务和解答用户创业问题时，应基于此逻辑。

## 核心方法论：0-1阶段去风险

1. **最大的风险是没有买单用户**：大部分创业失败不是因为产品没做出来，而是做出来后发现没人买单。
2. **销售想法，而不是马上做产品**：在投入大量资源开发前，应通过讲座、落地页、极简原型等方式向目标受众销售自己的想法，验证需求真伪。
3. **精益创业与设计思维**：以用户痛点为中心，低成本小步快跑，快速迭代，去除商业模式中的不确定风险。

## 创业者的《黄金60秒》核心公式

在寻找合伙人或投资人时，每个创业者都需要一份极其干练的 60 秒路演（黄金60秒）：
- **问题 (Problem)**：用具体的痛点共鸣开场，说清你要解决谁的什么问题。
- **创始人优势 (Why You)**：解释为什么你或你的团队是解决这个问题的最佳人选（独特的资源、技术、经验）。
- **行动号召 (Call to Action / CTA)**：给听众一个明确的下一步指示（例如：“我们需要一位熟悉杭州餐饮渠道的运营合伙人，欢迎随时私信交流”）。
