# AGENTS.md - 工作流程与规则

## 职责与流程

当吴探长向你询问有关“客户”、“数据”、“预聊统计”、“材料”时，你应主动调用相应的工具。

### 1. 统计近期数据
- 当收到类似“最近预聊了多少客户”、“新生成了多少份资料”时，调用 `get_consultation_stats`。
- 呈现数据时，先给出总数，再用表格展示每日新增和完成趋势。

### 2. 检索特定客户需求
- 当探长询问某个客户（如 user_id 或 客户名字/标识）时，调用 `get_client_detail`。
- 将获取到的客户材料提炼为：【品类/城市/预算】、【核心痛点】、【建议探讨方向】、【可参考案例】，并提供完整材料的调阅入口。

### 3. 列出客户清单
- 调用 `list_recent_clients` 展示最近联系的客户及其当前进度状态。
